# Pracownia TAK — wedding planning & eventy

Strona-wizytówka portfolio (fikcyjna marka) — butikowa pracownia wedding
planningu z Warszawy (Mokotów). Projekt pokazowy dla stronainternetowa.biz /
100strona.pl.

## Styl

Fashion-editorial romantyczny — jak zaproszenie ślubne haute couture
skrzyżowane z okładką magazynu: ogromna kursywa serif nachodząca na zdjęcia,
motyw „TAK." powtarzany typograficznie (marquee, watermarki), asymetryczne,
nachodzące na siebie kolumny, numeracja rozdziałów jak w programie ceremonii
(I.–VIII.), malutkie mono-podpisy przy wielkim serifie.

- **Paleta:** kość słoniowa `#FAF6F0`, czerń `#141210`, stary róż `#D8A7A0`,
  szałwia `#8A9B84`, złota linia `#B99A5B`
- **Fonty:** Cormorant Garamond (display, italic), Figtree (body), Cousine (mono)

## Sekcje

Hero („Powiedz TAK.") · marquee · O pracowni (zespół, od 2016) · Program
ceremonii (usługi) · Proces (5 aktów) · Pakiety z cenami · Realizacje (album) ·
Opinie par · FAQ · Kontakt z mapą · Stopka z creditem.

## Stack

Astro 5 + Tailwind CSS 4 (`@tailwindcss/vite`), zdjęcia Unsplash.

## Komendy

```bash
npm install
npm run dev      # podgląd lokalny
npm run build    # build produkcyjny do dist/
```

Hosting: Vercel (framework Astro wykrywany automatycznie).
